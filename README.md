## What is this?

**Gaia Awakening** is a kitchen-sink modpack built around **Create** as the central tech progression, enriched with automation (AE2, Mekanism), magic (Ars Nouveau, Apotheosis), exploration, and quality-of-life mods. It's designed for a small group of friends, with casual-friendly defaults and a guided quest system covering all five progression tiers.

The world generates with dramatically tall terrain (Tectonic + Terralith), underground rivers, and massively improved structure generation. You'll start punching trees and end up inside a flying fortress powered by a fusion reactor.

## Progression Tiers

| Tier | Theme | Key Mods |
|------|-------|----------|
| **T1 – The Beginning** | Survival, first machines, exploration | Create, Apotheosis, Artifacts, YUNG's |
| **T2 – Brass & Machines** | Automation, railways, storage | Create (Brass), Steam 'n' Rails, Sophisticated Storage |
| **T3 – Energy & Automation** | Digital storage, power networks, magic | AE2, Mekanism, Ars Nouveau, PneumaticCraft |
| **T4 – Advanced Systems** | Heavy industry, arcane crafting | Mekanism Generators, MEGA Cells, Mystical Agriculture |
| **T5 – Endgame** | Boss fights, flying machines, fusion | L'Ender's Cataclysm, Mekanism Fusion Reactor |

### The Catalyst System (Custom)

Progression into T3 and T4 seeds (Mystical Agriculture) is gated behind two custom items:

- **Resource Catalyst** — required for T3 seeds. Three acquisition paths:
  1. *Passive* — drops from Create Ore Excavation veins
  2. *Mechanical* — Mechanical Crafter recipe (MA essences + Apotheosis gem)
  3. *Active* — Catalyst Altar + Hyper Experience fluid (Create Enchantment Industry Spout)

- **Mythic Catalyst** — required for T4 seeds. Harder to obtain:
  1. *Boss drops* — Cataclysm Apex bosses only
  2. *Farm* — Drygmy mob farm (Ars Nouveau) + Catalyst Altar + Mythic Liquid XP

## Highlights

- **Tall World** — Tectonic with `max_y: 512`, `vertical_scale: 1.8`, underground rivers, rolling hills
- **Create-centric** — Create, Steam 'n' Rails, Create: New Age, Create: Enchantment Industry, Create Deco, and more
- **Deep storage** — AE2 + MEGA Cells (64M item cells), Sophisticated Backpacks/Storage, Functional Storage
- **Ars Nouveau** — full magic system including Ars Creo (Create integration), Ars Additions, Drygmy automation
- **Apotheosis** — gem socketing, enchantment overhaul, boss modifiers, spawn control
- **Exploration** — Alex's Mobs, Alex's Caves, Mowzie's Mobs, L'Ender's Cataclysm, When Dungeons Arise, all YUNG's mods
- **Custom items** — 15 unique items with hand-crafted textures (recolored from vanilla bases)
- **JourneyMap** — minimap and full map (FTB Chunks used only for chunk claiming)
- **Guided quests** — FTB Quests with 5 tier groups and cross-chapter dependencies

### XP Economy

- **Hyper Experience** (Create Enchantment Industry fluid) — base XP fluid, used in Catalyst Altar and Mixer
- **Mythic Liquid XP** (custom fluid) — 2000 mB Hyper XP + Singularity Shard → 1000 mB Mythic XP via heated Mixer
- Enables Mythic Catalyst crafting via Spout

## Installation

### Players

Download the latest `.mrpack` from the [Releases](../../releases) page and import it into your launcher (Modrinth App, ATLauncher, Prism Launcher, etc.).

**Recommended specs:**
- 8 GB RAM allocated minimum (12 GB recommended)
- GPU: GTX 1060 / RX 580 or better
- Java 21+

### Development

This modpack is managed with [packwiz](https://packwiz.infra.link/).

```bash
# Clone the repo
git clone https://github.com/VugasDev/gaia-awakening.git
cd gaia-awakening

# Add a mod
packwiz mr add <mod-slug>

# Refresh index after manual changes
packwiz refresh

# Export mrpack
packwiz modrinth export
```

**KubeJS scripts** live in `kubejs/server_scripts/` — numbered `01_` through `10_` for load order.

**Quest files** are in `config/ftbquests/quests/chapters/` as SNBT files. Use `&` escaping as `\&` in all string fields.

**CurseForge API key** goes in `~/.config/packwiz/.packwiz.toml` — never commit it.
